# Test Task DevRain  

In this repository you will find the solution for the "AI Engineer Test Task: Recipe Assistant" test task.
All files are divided into folders, corresponding in their names to the names from the task itself
Within the folders, all files specified in the "Deliverables" section of the task are named as specified there. 
All other files are named according to the same principle or for ease of use.

A dataset is required for tasks 3 and 4 to work correctly. The dataset used during testing is attached in the corresponding folders.
Tasks 3 and 4 require an embedding dataset. 
A portion of this dataset is included in the qdrant_db_local folders to demonstrate proper operation. 
If necessary, a new embedding dataset can be created by running the ingestion.py script in Tasks 3 or 4.
